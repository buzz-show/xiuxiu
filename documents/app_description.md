# app定义

## 1. 布局

app的布局分为两部分：底部导航栏和内容区。底部导航栏包含三个选项：首页、ai助手和我的。内容区根据用户选择的导航选项显示不同的页面。

**导航栏** ：撑满全宽。用苹果风格的tabbar，图标和文字垂直排列。选中状态下图标和文字变色。

## 2. 首页

首页展示用户最长关注的内容，包含以下模块：

- **宠物信息**：宠物头像，名称，体重。
- **今日状态**：宠物的睡眠时长、外出玩耍时长、进食重量、心情指数。
- **ai建议**：结合天气和心情指数，给出建议。
- **提醒**：宠物的驱虫提醒、洗澡提醒、疫苗提醒等。
- **最近洗护**：宠物的最近内驱日期，最近外驱日期，最近洗澡日期，上次打疫苗提醒。
